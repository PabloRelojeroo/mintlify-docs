Los sets de armadura otorgan bonificaciones especiales cuando equipas el **set completo** (casco, pechera, pantalones y botas).

## Sets disponibles

---

### Set Farmer

| Pieza | Material | Encantamientos |
|---|---|---|
| Casco | Cuero | Protección III, Durabilidad II |
| Pechera | Cuero | Protección III, Durabilidad II |
| Pantalones | Cuero | Protección III, Durabilidad II |
| Botas | Cuero | Protección III, Durabilidad II |

**Bonificación:** x1.5 FarmCoins en todo lo que farmees.

Ideal para sesiones de farmeo intensivo. Combínalo con un Booster de FarmCoins para maximizar ganancias. Puedes comprar las piezas individualmente en la sección **Especiales** de la `/tienda`.

| Pieza | Precio |
|---|---|
| Casco | 1.090 FC |
| Pechera | 1.680 FC |
| Pantalones | 1.520 FC |
| Botas | 1.100 FC |

## Cómo funcionan

- La bonificación **solo se activa con el set completo** equipado
- Si te quitas una pieza, pierdes el bonus inmediatamente
- Las bonificaciones se acumulan con boosters activos
- Cada set tiene nombre y descripción visible al pasar el cursor
