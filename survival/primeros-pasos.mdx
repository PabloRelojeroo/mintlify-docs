---
title: Primeros pasos
description: Qué hacer cuando entras por primera vez al Survival de Drakense.
---
Si acabas de entrar al servidor, esta guía te ayudará a entender cómo funciona todo.

## 1. Consigue tu kit inicial

Usa el comando `/kit` para abrir el menú de kits. El **Kit Inicial** está disponible para todos los jugadores y te dará:

- Herramientas de piedra
- 16 troncos de roble
- 16 de carne 
- una proteccion **10x10**
- Full set de cota de malla

<Note>
El kit inicial tiene un cooldown de 72 horas. Úsalo con cabeza.
</Note>

## 2. Empieza a farmear

Mata mobs y cosecha cultivos para ganar **FarmCoins**, una de las monedas del servidor. Con FarmCoins podrás comprar herramientas y materiales en la tienda.

| Mob/Cultivo | FarmCoins |
|---|---|
| Vaca | 2 |
| Cerdo | 2 |
| Gallina | 1 |
| Oveja | 2 |
| Conejo | 3 |
| Trigo, Zanahoria, Patata | 1 |
| Nether Wart, Cocoa | 2 |

## 3. Visita la tienda

Ejecuta `/tienda` o `/shop` para abrir la tienda del servidor. Ahí podrás gastar tus FarmCoins o Dinero en herramientas de distintos niveles, desde hierro hasta diamante encantado, bloques de decoracion, bloques utiles, redstone, y mas.

## 4. Forma o únete a un equipo

El survival se vive mejor en equipo. Usa `/team create <nombre>` para crear el tuyo o pídele a alguien que te invite con `/team invite`.

## 5. Dirígete a las arenas

Cuando tengas equipo y ganas de pelear, visita las arenas de combate. Por cada kill ganarás **Sangrias**, una moneda PvP que puedes usar para items exclusivos.

## Comandos esenciales

| Comando | Descripción |
|---|---|
| `/bal` | Ver tu balance de monedas |
| `/tienda` | Abrir la tienda |
| `/kit` | Abrir el menú de kits |
| `/team` | Gestionar tu equipo |
| `/nightvision` | Obtiene vision nocturna de manera gratuita |
