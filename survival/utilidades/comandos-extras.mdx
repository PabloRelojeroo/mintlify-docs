Comandos de utilidad que facilitan el día a día en el servidor.

## Ender Chest portátil

```
/ec
```

Abre tu Ender Chest desde cualquier lugar, sin necesidad de tener el bloque.

## Sombrero

```
/hat
```

Coloca el item que tengas en la mano como sombrero (slot del casco). Puramente cosmético.

## Cooldowns de items por zona

Algunos items tienen restricciones en ciertas zonas del servidor:

- **Ender Pearls** en spawn: cooldown de 10 segundos entre usos
- **Tótem de Inmortalidad** en spawn: máximo 2 usos

El cooldown restante se muestra en la barra de acción.

## XP optimizada

Los orbes de experiencia cercanos se fusionan automáticamente, reduciendo el lag y haciendo que recojas toda la XP de una vez.

## Resumen de comandos

| Comando | Descripción |
|---|---|
| `/fixhand` | Reparar item en mano |
| `/fixall` | Reparar todo el inventario |
| `/fly` | Volar en zonas seguras |
| `/ec` | Abrir Ender Chest |
| `/hat` | Poner item como sombrero |
| `/bal` | Ver balance de monedas |
| `/tienda` | Abrir la tienda |
| `/kit` | Menú de kits |
| `/team` | Gestionar equipo |
| `/frally` | Alerta de emergencia |
