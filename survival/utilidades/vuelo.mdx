# Vuelo en zonas seguras

Puedes volar dentro del spawn y en regiones protegidas donde seas dueño o miembro.

## Cómo activarlo

```
/fly
```

Usa el mismo comando para activar y desactivar el vuelo.

## Dónde funciona

| Zona | Vuelo permitido |
|---|---|
| Spawn | Si |
| Tu región protegida (dueño/miembro) | Si |
| Mundo abierto (sin protección) | No |
| Arenas | No |

## Restricciones

- Si sales de una zona permitida mientras vuelas, el vuelo se desactiva y caes al suelo
- No se puede usar durante combate

<Warning>
Ten cuidado al volar cerca del borde de una zona segura. Si cruzas el límite, caerás.
</Warning>
